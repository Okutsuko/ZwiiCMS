ZwiiCMS

Créez et gérez facilement un site web sans aucune connaissance en programmation.

Voir sur GitHub Télécharger Documentation

Ultra simple

Avec ZwiiCMS créer et maintenir votre site web devient un jeu d'enfant !

Sans base de données

ZwiiCMS fonctionne sans base de donnée, le contenu de votre site tient dans un seul fichier !

Super rapide

ZwiiCMS est léger et rapide, vos pages se chargeront aussi vite qu'une fusée !

Éditeur de texte

L'éditeur de texte de ZwiiCMS vous permettra une mise en page simple et efficace sans aucune connaissance en programmation !

Template responsive

ZwiiCMS utilise une structure de template moderne compatible PC, tablette et smartphone en même temps !

Open source

ZwiiCMS est complètement gratuit et open source, distribué sous licence GNU GPL v3.
